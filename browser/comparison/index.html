Performance Metrics Comparison

First Contentful Paint (FCP)

FCP measures how long it takes for the first piece of content to appear on your screen after starting to load a page.

  • ✅ Good: Under 1.8 seconds - Content appears almost instantly
  • ❌ Poor: Over 3.0 seconds - Users see a blank screen for too long

Largest Contentful Paint (LCP)

LCP measures when the largest content element (like a hero image or heading) becomes visible on the page.

  • ✅ Good: Under 2.5 seconds - Main content loads quickly
  • ❌ Poor: Over 4.0 seconds - Large elements take too long to appear

Interaction to Next Paint (INP)

INP measures how responsive your page is to user interactions (clicks, taps, and keyboard inputs).

  • ✅ Good: Under 200ms - Immediate response to user actions
  • ❌ Poor: Over 500ms - Noticeable delay in response

Cumulative Layout Shift (CLS)

CLS measures how much elements move around unexpectedly while the page loads, affecting visual stability.

  • ✅ Good: Under 0.1 - Content stays in place, no unexpected movements
  • ❌ Poor: Over 0.25 - Content jumps around, causing poor user experience

Total Blocking Time (TBT)

TBT measures how long the main thread is blocked, preventing user interactions during page load.

  • ✅ Good: Under 200ms - Page remains responsive during loading
  • ❌ Poor: Over 600ms - Page becomes unresponsive to user input

Time to First Byte (TTFB)

TTFB measures how long it takes for the server to respond with the first byte of data.

  • ✅ Good: Under 800ms - Quick server response time
  • ❌ Poor: Over 1.8s - Slow server response affecting overall page load
